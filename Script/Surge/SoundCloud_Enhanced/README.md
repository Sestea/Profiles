## SoundCloud Enhanced

解锁 SoundCloud Go+ 及其附带特性（如离线同步资料库等），去除导航栏和标签栏的 Neext Pro 推广图标。

```
https://raw.githubusercontent.com/Sestea/Profiles/master/Script/Surge/SoundCloud_Enhanced/SoundCloud_Enhanced.sgmodule
```

该模块需要解密 HTTPS 连接并进行修改，请在导入模块前检查 Surge CA 证书是否安装到系统并信任，MitM、脚本功能是否开启。

然后从以下 URL 导入并安装即可使用。
